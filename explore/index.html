The box below animates when the box within the scroll container comes fully into view. To the right is the CSS setting up this scroll-triggered animation:

                @keyframes fade-in {
                    from {
                      opacity: 0.1;
                      transform: scaleX(0.1);
                      background-color: black;
                    }
                  
                    to {
                      background-color: black;
                      transform: scaleX(3);
                    }
                }
                timeline-trigger: --viewtrigger view() contain 0% contain 100%;
                animation: fade-in .5s both;
                animation-trigger: trigger(--viewtrigger, enter play-forwards, exit play-backwards);

animation-play-state
animation-fill-mode
